# Airline-passenger-satisfaction-ml-analysis
Machine learning project analyzing 100k+ airline records using EDA and predictive models (Decision Tree, Random Forest) to identify key drivers of customer satisfaction.
Airline Customer Satisfaction Prediction

##  Overview

This project analyzes 100,000+ airline customer records to identify key factors influencing customer satisfaction. It uses Exploratory Data Analysis (EDA) and machine learning models to predict satisfaction levels.

##  Key Features

* Data cleaning and preprocessing
* Exploratory Data Analysis with visualizations
* Decision Tree and Random Forest models
* Feature importance analysis

##  Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

##  Results

* Random Forest achieved higher accuracy than Decision Tree
* Key drivers: service quality, delays, comfort
